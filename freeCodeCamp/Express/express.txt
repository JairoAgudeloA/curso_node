conceptos basicos de express:

crud:
operaciones básicas que se pueden realizar con la información almacenada.

Create // post
Read  // get
Update // put
Delete // delete


API:
application programming interface // interfaz de programación de aplicaciones

la api es un tipo de interfaz de software que permite que dos o mas programas se comuniquen entre sí.

nos permite desarrollar un software que ofrezca un servicio a otro software.

la api no es usada directamente por el usuario, sino por el programador que usa la api para implementar su programa.

normalmente cuando enviamos y recibimos informacion, trabajamos con el formato JSON.
